# Getting in Touch

We'd love to hear from you — whether you've found a bug, have a feature idea, or just want to talk about what you're building.

## GitHub Issues

The best way to report a bug or request a feature is to open an issue on GitHub:

**[github.com/graham-evans/AISandbox-Server/issues](https://github.com/graham-evans/AISandbox-Server/issues)**

When reporting a bug, please include:
- What you were doing when the problem occurred
- The version of AISandbox Server you are using
- Any relevant error messages or log output

## GitHub Discussions

For questions, ideas, and general conversation, use GitHub Discussions:

**[github.com/graham-evans/AISandbox-Server/discussions](https://github.com/graham-evans/AISandbox-Server/discussions)**

This is a good place to ask how to approach a simulation, share results, or suggest new directions for the project.
