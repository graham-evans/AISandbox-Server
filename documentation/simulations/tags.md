## Simulation Tags

As the number of simulations increases, we've introduced tags to make it easier to see what each does. The following tags have been defined:

###### Experimental

The simulation is not final and likely to change. Often this is due to balancing issues when the final scoring system has not been agreed.

**Don't expect results to be consistent between releases.**

###### Image

The simulation sends an image to the agents rather than a well-structured data object.

###### Random

The simulation includes random elements.

###### Single-Agent

The simulation is designed to run with a single agent.

###### Multi-Agent

The simulation runs with more than one agent.
